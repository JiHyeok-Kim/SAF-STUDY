7732. 시간 개념



개념 없는 은빈이 아 아니 시간 개념 없는 은빈이 때문에 문제 발생
-> 약속 시간 알려주는 프로그램 맹글어야 함
00 00 00 ~ 23 59 59
(숫자 2자리 아닐 경우 0으로 채움)
input으로 현재 시간과 약속 시간 받으면 => 남은 시간 출력
[현재 시간 보다 약속 시간이 더 빠른 경우 약속이 다음 날 있는 거]


방법[1] (보류)
0 input 받기
	string str;으로 입력 받고 parsing 함수 맹글어서 
	 그 함수에서 parsing 해줌 how ? 
	 str[인덱스]로 한 글자씩 접근해서
	 int형 변수에 -'0' 이걸로 숫자로 담아  
	 ":" 이거면 continue 하고
1 현재 시간이 약속 시간 보다 빠른 경우와 보다 느린 경우를 나눠서 함수로 호출
	(따로 나눠서 함수 맹글어야 나중에 trace 하기 용이함)
	원래는 23 > 00 인데 시간으로는 23 < 00 인거 인지 !
2 현재 시간이 더 빠른 경우 = 약속 시간 - 현재 시간 
	(xx xx xx 세 자리 모두 비교해서 빼 줘야 함)
3 현재 시간이 더 느린 경우 = 현재 시간 - 약속 시간
	(마찬가지로 세 자리 모두 비교해서 빼 주어야 함)

	 int형 변수에 -'0' 이걸로 숫자로 담아  
	 ":" 이거면 continue 하고

	1-1 >> cnt 변수 만들고 count 프로그램 만들어서 while 돌려 cnt++ 
2 2-1 >> ex 123456이면 / 10000 = 12,  / 100 =34, 남는 거 56
  2-2 >> 분하고 초는 /60 을 또 해서 올림까지 시키고 마무리
3 main에서 시, 분, 초  각각 출력 !


방법[3]
0 input 받기 string으로 받고 
1 parsing 함수 만들어서 strsub로 parsing & stoi 로 시, 분, 초로 나눠 받고
2 두 시간 다 초로 바꾸고 계산 
	2- 1 (if the current time is earliar than the appionment time)
